## Installation

1. Clone the repository:
```bash
git clone https://github.com/merry-berry-acai/client.git
cd client
```

2. Install dependencies:
```bash
npm install
```

3. Create `.env` file:

```bash
VITE_API_URL=http://localhost:5000/api
VITE_STRIPE_KEY=your_stripe_key
```

1. Start development server:
```bash
npm start
```
